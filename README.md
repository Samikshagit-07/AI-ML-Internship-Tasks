DATASET ANALYSIS REPORT
Task 1: Data Understanding and Statistical Summary
1. Data Structure Overview

The Titanic dataset consists of 891 observations (rows) and 12 features (columns).

The dataset includes a mix of numerical, categorical, and ordinal data types.

2. Variable Identification

Target Variable: Survived (The primary variable we aim to predict).

Numerical Features: Age, Fare, SibSp (Siblings/Spouses), and Parch (Parents/Children).

Categorical Features: Sex, Embarked, Pclass (Ticket Class), and Cabin.

3. Missing Value Analysis

Age: Contains 177 missing values, which represents about 20% of the data.

Cabin: Contains 687 missing values, making it the most incomplete column.

Embarked: Contains 2 missing values.

4. Statistical Summary & Insights

The average age of passengers was approximately 29.7 years, with the oldest being 80.

The overall survival rate in the dataset is approximately 38%.

Visualization shows that female passengers had a significantly higher survival count than male passengers.
